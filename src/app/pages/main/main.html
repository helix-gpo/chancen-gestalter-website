Häufig gestellte Fragen

Haben Sie Fragen?

Hier finden Sie schnelle Antworten auf die häufigsten Fragen. Für weitere Informationen stehen wir Ihnen gerne persönlich zur Verfügung.

Wie lange dauert der Anerkennungsprozess für ausländische Qualifikationen?

@if (open1) {

Wie lange die Anerkennung ausländischer Qualifikationen dauert, hängt von Abschlussart, Bundesland und zuständiger Stelle ab - in der Regel 3 bis 8 Monate.

}

Welche Kosten entstehen für Ihre Dienstleistungen?

@if (open2) {

Das Erstgespräch ist immer kostenlos. Die weiteren Kosten richten sich nach dem individuellen Leistungsumfang. Wir erstellen Ihnen ein transparentes Angebot ohne versteckte Kosten.

}

In welchen Sprachen können Sie beraten?

@if (open3) {

Wir bieten mehrsprachige Beratung an, unter anderem in Deutsch, Englisch und Arabisch - weitere Sprachen je nach Bedarf. Kontaktieren Sie uns gerne, um Ihre bevorzugte Sprache zu besprechen.

}

Können Sie auch bei rechtlichen Fragen unterstützen?

@if (open4) {

Wir bieten keine eigene Rechtsberatung an, beziehen diese falls erforderlich ein, um Sie optimal zu unterstützen.

}

Wie läuft die Patientenbegleitung für medizinische Behandlungen ab?

@if (open5) {

Wir organisieren die gesamte Reise - von der Klinikauswahl über Visa und Unterkunft bis zur Nachsorge. Sie erhalten einen persönlichen Betreuer, der Sie während des gesamten Aufenthalts begleitet und sprachlich unterstützt.

}
